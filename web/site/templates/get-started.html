Pyx is not yet ready for normal developers to write games with, but it is in active development and we are getting closer to our first public release with each day passing. Therefor, some manual installation and configuration is necessary to get up and running.

Additionally, some familiarity with the Common Lisp programming language is required, as well as the installation of a code editor that supports the interactive workflow of the language, and any additional tooling infrastructure.

Getting Familiar with Common Lisp

We recommend picking up a Common Lisp book and learning more about this truly powerful language. There are many great resources available. If you are on a tight budget, we recommend reading Practical Common Lisp for free online. Additionally, The Common Lisp Cookbook is another free valuable resource.

Installing a Development Environment for Common Lisp

Pyx requires a code editor with rich interactive development support for Common Lisp. Currently, we can only recommend Emacs to fill this role. For a quick and painless pre-configured version of Emacs with Common Lisp support, there is Portacle. For more information on getting up and running with Common Lisp, consult the Getting started and Editor support sections of The Common Lisp Cookbook.
